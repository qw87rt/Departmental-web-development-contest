Program Descriptions

Bachelor of Science in Computer Science - it is a four-year undergraduate program which covers the fundamentals of computer systems and operations. The program prepares the students to handle and form algorithms for solving computational problems. Students are expected to learn how to design, modify, and write software components and applications that constitutes the solution to software issues.

Bachelor of Science in Information Technology - it is a four-year undergraduate program that prepares students to become proficient on IT applications and will learn how to administer, operate, manage, and maintain information technologies. It emphasizes learning of both software and hardware management, and on strong database management skills.

Associate in Computer Technology - it is a two-year undergraduate program that prepares students to learn the necessary skills required for entry-level positions in the IT industry. You may also proceed to either CS or IT programs if you attain the requirements that comes with finishing ACT.

The school branches the program into two majors which are networking and applications development.
